Flexbox Mixins

The Flexbox Layout module provides a more efficient way to construct layouts. The core idea behind it is to allow the container to alter the width, height and order of it's child elements to maintain the expected layout. That is, the container can expand items to fill the available space or shrink them to prevent overflow.

    See the links below for a more information:
  • Using CSS flexible boxes
  • A Complete Guide to Flexbox
  • Flexbox adventures

Bourbon provides an exhaustive set of flexbox mixins. Below are a few examples which demonstrate how to use these mixins:

/* This is an example of a basic grid layout */

/* A row container */
.flex-row {
  @include display(flex);
  width: 100%;
}
/* Clearfix */
.flex-row:before,
.flex-row:after {
  content: "";
  display: table;
  clear: both;
}
/* Flexible column */
.flex-row__col {
  @include flex(1);
  display: block;
  width: 100%;
}
/* Columns with initial sizing */
.flex-row__col--25 {
  @include flex(0);
  @include flex-basis(25%);
  max-width: 25%;
}
.flex-row__col--fixed {
  @include flex(0);
  @include flex-basis(250px);
  max-width: 250px;
}

/* Alignment example */

/* .flex-row__col-* vertically aligns an individual .flex-row__col */
.flex-row__col--top {
  @extend .flex-row__col;
  @include align-self(flex-start);
}
.flex-row__col--center {
  @extend .flex-row__col;
  @include align-self(center);
}
.flex-row__col--bottom {
  @extend .flex-row__col;
  @include align-self(flex-end);
}

/* Offset columns */
.flex-row__col--offset-25 {
  margin-left: 25%;
}
.flex-row__col--offset-33 {
  margin-left: 33.3333%;
}
